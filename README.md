# ☕ Java Programming I - University of Helsinki

This repository contains my solutions to the exercises from the comprehensive **Java Programming I** course by the University of Helsinki.

The goal of this project is to master the fundamentals of programming, algorithms, and Object-Oriented Programming (OOP) using Java.

## 📁 Repository Structure
The project is structured into parts, tracking my learning progress:
* **Part 1:** Conditionals, variables, and basic logic.
* **Part 2:** Loops, methods, and breaking problems down.
* **Part 3:** Lists, arrays, and string manipulation.

## 🛠️ Tools & Technologies
* **Language:** Java 11
* **Environment:** TMC (Test My Code) extension with NetBeans/IntelliJ
* **Version Control:** Git & GitHub
