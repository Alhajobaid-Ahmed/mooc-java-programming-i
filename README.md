# University of Helsinki Java MOOC - Part 1

This repository contains my solutions to the exercises from the famous **Object-Oriented Programming with Java** course offered by the University of Helsinki. 

The purpose of this project is to build a solid foundation in Java, core programming concepts, and Object-Oriented Programming (OOP) principles.

## 🚀 Technologies Used
* **Java** (JDK 11+)
* **TMC (Test My Code)** framework for exercise validation
* **NetBeans / IntelliJ IDEA** as the IDE

## 📂 Project Structure
The repository is organized by course parts:
* `part01/` - Basic variables, conditionals, and loops.
* `part02/` - Methods, dividing code, and recurring problems.
* `part03/` - Lists, arrays, and strings.

## 🧠 What I Learned
* Writing clean and readable Java code.
* Implementing loops, conditionals, and data structures (ArrayLists, Arrays).
* Understanding the basics of Object-Oriented Programming.
